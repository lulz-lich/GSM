# GSM Usage

## CLI

Backup:
```bash
./src/gsm_cli.sh backup
```

Restore:
```bash
./src/gsm_cli.sh restore
```

Sync cloud to local:
```bash
./src/gsm_cli.sh sync
```

Manual backup:
```bash
./src/gsm_cli.sh manual-backup "/path/to/save" "Game Name"
```

## TUI

```bash
./src/gsm_tui.sh
```

## GUI

```bash
./src/gsm_gui.py
```
